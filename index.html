Hacker News

Started as a project to play with Square's Retrofit library, ended up making a daily driver for reading hacker news.

Uses cheeaun's node-hnapi project for it's API backend.

Building

Copy each of the *.properties.example to their respective properties files and fill out as necessary.

Import into Android studio, should work out of the box.
